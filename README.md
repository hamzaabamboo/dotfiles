# Dotfiles

My own terminal configuraition using zsh and zgen

## Prerequisits

- `zsh`
- `zgen`

## Installation

- clone this repo to home folder
- `ln -s dotfiles/.zshrc ./zshrc`
- `source ~/.zshrc`
